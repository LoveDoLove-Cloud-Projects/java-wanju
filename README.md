## 参数:vls,vms,spl,xhttp,rel,hy2,tuic，sock,3x含义:

vls即vless+ws+tls

vms即vmess+ws+tls

xhttp即vmess+xhttp+tls

rel即reality

hy2即hysteria2

3x即3种协议，包括vmess,tuic,hy2
